NullJobs - Automated Jobs System for Spigot/Paper

Region-based AutoMine, AutoForest & AutoFish system for Minecraft servers.

NullJobs is a lightweight but powerful jobs plugin built around WorldGuard regions, allowing players to interact with automated region-based jobs such as mining, forestry, fishing and milking.

Supports economy rewards, automatic regeneration, animations, and optional holograms.

FEATURES

✔ AutoMine Regions
Automatic regenerating mines with configurable ore generation and reward systems.

✔ AutoForest Regions
Hit-based tree chopping with immersive falling animations and automatic restoration.

✔ AutoFish Regions
Custom fishing regions with configurable catches and rewards.

✔ AutoMilking Regions
Interactive cow milking jobs with progress system and cooldowns.

✔ WorldGuard Integration
Everything works inside protected WorldGuard regions.

✔ DecentHolograms Support
Optional holograms with live mine refresh countdowns.

✔ Lightweight & Optimized
Designed for modern Spigot/Paper servers with minimal performance impact.

AUTOMINE

Automated regenerating mines.

Players mine ores while the mine refreshes automatically after a configurable timer.

Features:

• configurable ore chances
• vanilla drops or money rewards
• optional hologram timer
• refresh intervals
• replace-block whitelist

Perfect for:

• Prison
• Survival Economy
• Skyblock
• RPG

━━━━━━━━━━━━━━━━━━━━━━

AUTOFOREST

Interactive tree chopping.

Players hit logs multiple times until the tree collapses and restores automatically.

Features:

• hit-based chopping
• leaves ignored
• falling tree animation
• no falling-block trash
• automatic restore

Rewards:

• items
• money

Perfect for:

• Survival
• RPG
• Farming zones

Rewards:

WHY NULLJOBS?

Unlike traditional jobs plugins, NullJobs focuses on interactive region-based gameplay systems.

Instead of repetitive grinding, players interact with real environments:

• regenerating mines
• animated forests
• custom fishing zones

Simple to configure, lightweight, and built for modern Minecraft servers.
